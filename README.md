# Mizunaとは

Mizunaは、ノート時に無駄な通信を行わず通信量を削減したい方向けのノート専用Misskeyクライアントです。

## 公式サーバー

- [安定版（推奨）](https://mizuna.blossomsarchive.com/): 安定したバージョンのMizunaを利用できます。
- [開発版](https://mizuna-canary.blossomsarchive.com/): 最新の機能を試すことができますが、不安定な場合があります。
